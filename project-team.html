Project Overview

This eCommerce website project demonstrates modern web development practices using HTML5, CSS3, and vanilla JavaScript enhanced with GSAP animations. The site features a fully functional shopping cart with PKR pricing, product filtering, responsive design, form validation, smooth animations, checkout flow, and order confirmation — all built with premium aesthetic design.
